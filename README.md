# Unityfs-Searcher
It's a highly optimised python script for Shadow Fight 3 and Shadow Fight Arena, usefull in searching unityfs bundles for some asset name. 
1. REQUIRES A PYTHON INTERPRETOR 
2. Uses Mipmap and threading to get a average search time of ~5 seconds in 3gb worth file.
3. Has hardcoded folder address in open readable and writable location, user can change it as required

  Installation-
  pip install rich
